Papel: Desenvolvedor Backend

Resultados: API pública open source, focada em praticidade para desenvolvedores brasileiros, com endpoints úteis e documentação clara.

InfoHub API — Centralize informações públicas do Brasil com FastAPI!
Uma API pública que reúne dados práticos do dia a dia em um só lugar:

  • Endereço via CEP
  • Clima atual por cidade
  • Cotações de moedas (USD, EUR etc.)
  • Feriados nacionais por ano

Interface RESTful simples, rápida e bem documentada — ideal para integrar em projetos web, bots, apps ou estudos.

Exemplos de uso:

  • /cep/01001000 → Praça da Sé, São Paulo - SP
  • /clima?cidade=Recife → 28.6°C, nuvens dispersas
  • /cambio → Cotações de dólar e euro
  • /feriados?ano=2025 → Lista de feriados nacionais

100% open source. Simples, útil e direto ao ponto.
